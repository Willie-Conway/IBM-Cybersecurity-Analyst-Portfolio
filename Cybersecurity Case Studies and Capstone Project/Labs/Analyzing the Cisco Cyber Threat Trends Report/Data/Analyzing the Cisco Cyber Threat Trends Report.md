# Hands-on Lab: Analyzing the Cisco Cyber Threat Trends Report

**Estimated time:** 30-45 minutes

---

## Project Scenario

You're a **Junior Security Analyst** at **Nexus Defense Solutions**, a cybersecurity consulting firm serving mid-sized enterprises across North America.

Your manager has asked you to review the latest **Cisco Cyber Threat Trends Report** based on DNS activity observed by Cisco Umbrella. The leadership team needs a clear understanding of the current threat landscape to prioritize security investments for the upcoming fiscal year. They've specifically asked you to:

1. Identify the top threats organizations are facing
2. Understand how these threats operate and evade detection
3. Recommend defensive controls based on the report's findings

Your analysis will directly influence the company's security strategy and budget allocation for the next 12 months.

## Answer Key (Based on Typical Cisco Report Findings)

> **Note:** Answers will vary depending on the specific report version. Use these as guidance.

### Part 2: Threat Definitions (Example)

| Threat Category      | Definition                                                                                                                              |
| :------------------- | :-------------------------------------------------------------------------------------------------------------------------------------- |
| Information Stealers | Malware designed to secretly collect sensitive data including credentials, financial information, and personal identifiable information |
| Trojans              | Malicious software disguised as legitimate applications that trick users into installing them                                           |
| Ransomware           | Malware that encrypts files and demands payment for decryption keys                                                                     |
| RATs                 | Remote Access Trojans that provide attackers with backdoor control over infected systems                                                |
| APTs                 | Advanced Persistent Threats - long-term, targeted attacks focused on espionage and intellectual property theft                          |
| Botnets              | Networks of compromised devices controlled remotely to conduct large-scale attacks                                                      |
| Droppers             | Malware designed to deliver and install other malicious payloads                                                                        |

### Part 6: Why Threats Persist (Example)

| Factor               | Explanation                                                                              |
| :------------------- | :--------------------------------------------------------------------------------------- |
| Profitability        | Cybercrime has low risk and high financial return compared to traditional crime          |
| Stealth              | Modern malware uses encryption, living-off-the-land techniques, and legitimate services  |
| Adaptability         | Threat actors quickly change tactics based on security improvements                      |
| Low barrier to entry | Malware-as-a-service and exploit kits make cybercrime accessible to non-technical actors |

## Key Takeaways

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    KEY TAKEAWAYS FROM CISCO REPORT                           │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   1. INFORMATION STEALERS are the most prevalent threat                     │
│      → They target credentials and financial data                          │
│      → Often delivered via phishing                                        │
│                                                                              │
│   2. RANSOMWARE continues to evolve                                        │
│      → Double extortion (encryption + data leak)                          │
│      → Ransomware-as-a-service (RaaS) models                               │
│                                                                              │
│   3. DNS-LAYER SECURITY is critical                                        │
│      → Blocks communication before connection established                  │
│      → Effective against malware that evades other controls                │
│                                                                              │
│   4. THREAT ACTORS ADAPT QUICKLY                                           │
│      → Continuous monitoring required                                      │
│      → Static defenses become obsolete                                     │
│                                                                              │
│   5. LAYERED DEFENSE is non-negotiable                                     │
│      → No single control stops all threats                                 │
│      → Prevention + Detection + Response                                  │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```
